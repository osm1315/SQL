## 대표적인 SQL 문법
1. COUNT: 행수를 확인, 의도한 데이터의 행 개수가 맞는지 확인 
2. NOT NULL: 특정 컬럼에 NULL이 존재하는지, 필수 필드가 비어있는지 확인 
3. DISTINCT: 데이터의 고유값을 확인해 중복 여부 확인 
4. IF문, CASE WHEN: 의도와 같다면 TRUE, 아니면 FALSE 
5. 1,3번을 활용해 두 컬럼을 보고 갯수 비교
